Internsctl \- Custom command line script
.SH SYNOPSIS
.B internsctl[\fI\,OPTION\/\fR]...

.SH DESCRIPTION
.PP
Commands to get system,files,user information and manage them and provides various functionalities to perform operations on the Linux system.
.SH OPTIONS

.TP
\fB\--version\fR
Print version info.
.TP
\fB\--help\fR
Povide help for using internsctl commands.
.TP
\fB\--author\fR 
Print the author of each file.

.SH COMMANDS

\fBcpu getinfo\fR 
Print cpu information of my server
.TP
\fBmemory getinfo\fR
Print memory information of my serve

.TP
\fBuser create <username>\fR
Create a new user on my server who can login to linux system and access his home
directory
<username> should be replaced by new username
.TP
\fBuser list [\fI\,OPTION\/\fR]\fR
Print the list of all the regular users present on my server

\fBOPTIONS\fR

\fB\--sudo-only 
: Print all the users with sudo permissions
.TP
\fBfile getinfo [options] <file-name>\fR
Print info about file

\fBOPTIONS\fR

\fB\--size , -s\fR
: Print size

\fB\--permissions , -p\fR
: Print file permissions

\fB\--owner , -o\fR
: Print file owner

\fB\--last-modified , -m\fR
: Print last modified

.TP
0

.TP

.TP
2
